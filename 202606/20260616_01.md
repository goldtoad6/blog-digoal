## 背景  
  
曾经的数据库, 给应用准备的, 面向确定性:  
- 高效存
- 高效读
- 不丢数
  
未来的数据库, 给 Agent 准备的, 面向不确定:
- 自动整理; 数据绝对不是存进来怎么样就是怎么样, 应该有时序关联、语义关联、网状图式关联. 就像我们的大脑, 后台不断整理每天接触到的信息.
- 自动遗忘整理; 该忘的忘.
- 自动沉淀 skill; 根据交互, 分析意图, 沉淀出 skill
- 进化模型; 小脑(数据库内置本地专用小模型, 负责整理、遗忘、沉淀 skill)
- mcp 是标配(存疑)
  
如果你想吃 agent 数据库市场, 自我进化是必备能力, 否则你就只能守传统市场.
