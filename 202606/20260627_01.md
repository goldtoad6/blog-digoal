## 背景  
  
  
如果你想从一段视频、一场直播、一段播客的音频中提取文字, 首先需要录制下音频, 然后采用 ASR 音频转文字大模型将其转换成文字.  
  
利用 macOS 自带的 QuickTime Player，再配合一个免费工具 BlackHole 来实现的方案。需要简单设置，但完全免费。  
  
1、安装虚拟音频驱动：下载并安装一个名为 BlackHole 的免费开源软件。它的作用就像一个虚拟的音频线，能把电脑内部的声音“转发”给录音软件。  
  
brew 配置可参考: [《珍藏级 - Claude , Codex 使用经验总结》](../202604/20260415_05.md)  
  
```  
HTTP_PROXY=http://127.0.0.1:22222 HTTPS_PROXY=http://127.0.0.1:22222 ALL_PROXY=http://127.0.0.1:22222 brew install blackhole-2ch  
```  
  
2、打开 “音频 MIDI 设置” 应用 APP，创建一个 “多输出设备”，同时勾选你实际的扬声器和 BlackHole。然后在系统设置 “声音” 里，将输出设备选择为你刚创建的这个多输出设备。这样你既能听到声音，又能让系统声音流向 BlackHole。  
  
3、使用 QuickTime Player 录制：打开 QuickTime Player，选择“新建屏幕录制”，在录制选项的麦克风下拉菜单中，选择 BlackHole 作为音频输入源。这样录制下来的视频就会包含系统内部的声音了。  
  
4、开始录制后, 开启要录制音频的内容, 例如“一段视频、一场直播、一段播客”.  
  
5、录制完成后, 将 QuickTime Player 输出的 m4a 音频文件转换成 mp3 格式.  
  
ffmpeg 配置可参考: [《使用 ffmpeg 将图片+音频合成视频》](../202605/20260506_01.md)  
```  
ffmpeg -i input.m4a -c:a libmp3lame -b:a 320k output.mp3  
```  
  
6、使用 ASR 音频转文字大模型, 提取 mp3 中的文字.  
  
例如百炼的 Qwen3-ASR-Flash  
  
https://bailian.console.aliyun.com/cn-beijing?spm=5176.29597918.0.d_primary.4b34133cu5iPda&tab=model#/efm/model_experience_center/voice?currentTab=voiceAsr  
  
或者 step 提供的 asr 模型 , 薅 step 羊毛参考:  
  
https://mp.weixin.qq.com/s/808uFGyhJSEUMMpsTQsxJA  
  
https://mp.weixin.qq.com/s/0NWdufFfBx8cGNUyhKzU7Q  
  
7、或者也可以本地 ASR 大模型(免费, 且不用出网络私密性好)  
  
```
git clone --depth 1 https://github.com/ggerganov/whisper.cpp.git
cd whisper.cpp

# 下载模型权重（注意: 这步是从 huggingface 拉取转换好的 ggml 格式)
https://huggingface.co/ggerganov/whisper.cpp/resolve/main/ggml-large-v3.bin
https://huggingface.co/ggerganov/whisper.cpp/resolve/main/ggml-medium.bin

mv ~/Downloads/ggml-large-v3.bin models/
mv ~/Downloads/ggml-medium.bin models/

# 如果你使用的是 apple M 系列芯片, whisper.cpp 有 GPU 加速, 建议使用 ggml-large-v3.bin (输出简体)
# 如果你使用的是 intel 系列芯片, whisper.cpp 无 GPU 加速, 建议使用 ggml-medium.bin (输出繁体)

# 编译（会自动启用 Metal 加速）
brew install --formula cmake
make

# 把 m4a 转成 16kHz wav（whisper.cpp 需要这个格式）
ffmpeg -i input.m4a -ar 16000 -ac 1 -c:a pcm_s16le output.wav

# 转录
./build/bin/whisper-cli -m models/ggml-large-v3.bin -f output.wav -l zh -otxt

# 如果 output.wav.txt 是繁体
# 繁体 转 简体
brew install opencc
opencc -i output.wav.txt -o output_simplified.txt -c t2s.json
```
