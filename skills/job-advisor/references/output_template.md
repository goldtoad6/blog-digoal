# Markdown 输出模板

## 文件命名

`<cwd>/markdown/<YYYY-MM-DD>_job-recommendations.md`

例:`/Users/digoal/new/markdown/2026-06-16_job-recommendations.md`

## 模板正文

```markdown

- 报告生成时间:2026-06-16 15:30
- 抓取平台数:8
- 抓取原始条目:30 → 去重后 24 → 通过打分排序后保留 10

> ⚠️ 数据通过公开搜索引擎聚合,可能滞后或不完整,链接请人工二次核实后投递。

| 2 | ... | ... | ... | ... | ... | ... | ... |

## 二、逐条详情

### #1 数据工程兼职 — XX科技
- **地点**:远程
- **薪资**:250 元/时
- **工作形式**:兼职 / 远程
- **技能匹配**:1.0(Python / PostgreSQL / Airflow 全命中)
- **通勤折算**:1.0(远程)
- **灵活度**:1.05
- **类型匹配**:1.0
- **性价比分**:1.85
- **JD 摘要**:...(原 snippet 摘录)
- **链接**:https://...
- **风险提示**:薪资描述来自 snippet,建议投递前核实结算方式(时薪/项目/天)

### #2 ...

## 三、被过滤/降权条目摘要

- 5 条因关键词命中"销售"被降权至底部
- 3 条因 `pay` 无法解析标记为 `pay_unknown=true`,未进入 Top10
- 2 条因与 `min_hourly_rate=150` 不符被标记 `meets_salary_floor=false`

## 四、数据来源与免责声明

- 抓取时间:2026-06-16 15:30

  - 投递前请务必打开链接核实岗位状态与薪资条款
  - 本报告由 LLM 自动生成,不构成就业推荐或法律建议
  - 远程/兼职岗位请警惕押金、培训费、刷单等常见骗局
```

## 渲染要点

1. **Top10 表格**:严格 8 列,标题不要换行,链接用 `<a name>` 锚点跳转
2. **逐条详情**:
   - 必含:地点 / 薪资 / 工作形式 / 四项系数 / 性价比分 / JD 摘要原文 / 链接
   - 选含:风险提示(对远程、薪资虚高、来源豆瓣/V2EX 等社区类)
3. **被过滤摘要**:让用户知道为什么有些岗位没进 Top10,避免他重复问
4. **免责声明**:必须有,平台数据滞后,法律风险提示

## 输出位置确认

```python
from pathlib import Path
from datetime import date
out_dir = Path.cwd() / "markdown"
out_dir.mkdir(exist_ok=True)
out_path = out_dir / f"{date.today().isoformat()}_job-recommendations.md"
```

若 `Path.cwd() / markdown` 不存在,先 `mkdir -p`。
