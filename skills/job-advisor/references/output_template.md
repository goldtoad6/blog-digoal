## 渲染要点

1. **Top10 表格**:严格 8 列,标题不要换行,链接用 `<a name>` 锚点跳转
2. **逐条详情**:
   - 必含:地点 / 薪资 / 工作形式 / 四项系数 / 性价比分 / JD 摘要原文 / 链接
   - 选含:风险提示(对远程、薪资虚高、来源豆瓣/V2EX 等社区类)
3. **被过滤摘要**:让用户知道为什么有些岗位没进 Top10,避免他重复问
4. **AI 友好专区**:
   - 独立小节呈现,**不并入主排名**
   - 排序:已知时薪降序 > 未知薪资靠后
   - 每条必含:**命中关键词** + **AI 工具链建议**(让用户知道用什么 AI 提效)
   - 锚点用 `<a name="ai-N">` 与 Top10 的 `<a name="N">` 区分
5. **免责声明**:必须有,平台数据滞后,法律风险提示
