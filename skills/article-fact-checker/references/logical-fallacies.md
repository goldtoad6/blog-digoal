# 常见逻辑谬误清单(L2 段落层排查用)

排查方式:每段扫一遍,命中即标 `[谬误:xxx]`。

## 一、形式谬误(逻辑结构本身有问题)

| 谬误 | 定义 | 典型句式 |
|---|---|---|
| 循环论证 | 用结论证前提 | "A 是真的,因为 A 为真" |
| 肯定后件 | 若 P→Q 且 Q,则 P | "下雨地湿,地湿所以下雨" |
| 否定前件 | 若 P→Q 且 ¬P,则 ¬Q | "鸟会飞,蝙蝠不是鸟,所以不会飞" |
| 合成谬误 | 个体属性推到整体 | "A 富,所以 A 国富" |
| 分割谬误 | 整体属性推到个体 | "A 国富,所以每个 A 国人富" |

## 二、相关性谬误(把不相关当相关)

| 谬误 | 定义 | 典型句式 |
|---|---|---|
| 诉诸权威 | 未给权威资质或反例 | "爱因斯坦说,所以对" |
| 诉诸传统 | "一直如此" | "我们祖辈都这样" |
| 诉诸大众 | "大家都这么认为" | "99% 的人都…" |
| 诉诸新颖 | "最新研究所以对" | "2024 年新发现…" |
| 诉诸无知 | 没人证伪就是真 | "没人能证明不存在" |
| 诉诸自然 | 自然的=好的 | "天然的就是好的" |
| 诉诸情感 | 用情绪代替论证 | "想想孩子们!" |
| 个例代替总体 | 单个例子代表全部 | "我朋友 X 怎样,所以…" |

## 三、因果谬误

| 谬误 | 定义 | 典型句式 |
|---|---|---|
| 后此谬误 | A 在 B 之前,A 导致 B | "我戴帽子球队赢了,所以…" |
| 混淆相关与因果 | X、Y 同时变,X→Y | "冰激凌销量和溺水率正相关" |
| 忽略第三变量 | 隐藏共同原因 | "学历高收入高"(可能是 IQ) |
| 单因果谬误 | 多因归一 | "成功靠努力"(忽略运气、资源) |
| 滑坡谬误 | 无依据的链式推断 | "允许 X,必然导致 Y,必然 Z…" |

## 四、统计谬误

| 谬误 | 定义 | 典型句式 |
|---|---|---|
| 幸存者偏差 | 只看成功者 | "成功创业者都不爱上学" |
| 样本偏差 | 样本不具代表性 | "我们班 50 人调查" |
| 基率忽视 | 不看总体概率 | "阳性检测后医生忽视 99% 阴性基率" |
| 辛普森悖论 | 分组/整体趋势相反 | 男女录取率都高,合计反而低 |
| 赌徒谬误 | 独立事件"该来了" | "连输 5 把,下一把必赢" |
| 抽样陷阱 | 自选样本 | "我采访的 10 个专家" |

## 五、结构谬误

| 谬误 | 定义 | 典型句式 |
|---|---|---|
| 稻草人 | 曲解对方再反驳 | "对方说 A,所以他支持极端 A'" |
| 红鲱鱼 | 引入无关议题转移 | 问到 X,答 Y |
| 虚假两难 | 只给两个极端选项 | "要么 A 要么 B" |
| 完美风暴 | 列举全部最坏情况 | 实际概率极低 |
| 偷换概念 | 关键词义偷换 | 同一词前段后段不同义 |
| 范围谬误 | 量词作用域错 | "所有人都有时候…" |

## 六、证据呈现谬误

| 谬误 | 定义 | 典型句式 |
|---|---|---|
| 选择性呈现 | 只用支持数据 | 忽略反面证据 |
| 锚定 | 第一个数字定基调 | 先说大数再说小 |
| 框架效应 | 同一事实不同框架 | "90% 存活" vs "10% 死亡" |
| 加载语言 | 用情感词代替判断 | "恐怖主义" vs "武装抵抗" |
| 伪精确 | 不必要的精度 | "73.83%" |

## 七、来源谬误

| 谬误 | 定义 | 典型句式 |
|---|---|---|
| 引用错位 | 引文与作者原意不同 | — |
| 二手转引 | 转了几手,源头失真 | "据 XX 转述 YY 说…" |
| 匿名权威 | 不具名"专家" | "一位不愿透露姓名的…" |
| 同行评议冒充 | 实际未经评议 | — |

## 实操建议

不要每段都查全部 80+ 谬误。按文章定调优先级排查:

| 文章定调 | 优先排查 |
|---|---|
| 评论 / 专栏 | 诉诸情感 / 稻草人 / 滑坡 / 虚假两难 |
| 营销文 | 选择性呈现 / 锚定 / 加载语言 / 伪精确 |
| 调查报道 | 二手转引 / 引用错位 / 匿名权威 |
| 学术 | 样本偏差 / 基率忽视 / 后此谬误 |
| 宣传 / 政治 | 诉诸大众 / 诉诸情感 / 偷换概念 / 框架效应 |

## 输出

每命中一个,在段落批注中记录:

```
[段落 X] 谬误: <类型> — 原文: "..."
严重度: 高(直接削弱论证) / 中(影响可信度) / 低(修辞瑕疵)
```
