# L2 段落层审查清单

段落层只解决:**每个段落主张什么?段落间是否互相支撑?推理结构是否成立?**

## 段落结构 5 项检查

### 1. 主题句存在性

每段第一句或末句应有一个可识别的核心论点。

- ☐ 全部段落都有明确主题句
- ☐ 部分段落主题句含糊
- ☐ 多段无主题句(红旗 — 结构混乱或凑字数)

### 2. 单段单一论点

一段一个核心论点。混杂多个论点的段落通常不可靠。

标记方式:每段顶部批注 `[单点] / [多点混杂]`。

### 3. 段落间关系矩阵

对每对相邻段落标注关系:

| 关系 | 说明 | 红旗信号 |
|---|---|---|
| 递进 | 后段加深/限制前段 | 正常 |
| 并列 | 多个角度支持同一主张 | 正常 |
| 对比 | 提供反例或对立视角 | **加分**(体现严谨) |
| 回扣 | 后段呼应开篇立论 | 正常 |
| 游离 | 与立论无明显关联 | **红旗**(凑字数 / AI 文典型特征) |
| 矛盾 | 后段否定前段 | **严重红旗** |

### 4. 段落与立论回扣

每段最终是否服务于 L1 立论?

- ☐ 所有段落都回扣立论
- ☐ 有 1-2 段游离(可接受,标注即可)
- ☐ ≥3 段游离(红旗 — 可能非原创/拼凑)

### 5. 推理类型标注

每段标出主要推理方式:

| 推理类型 | 强度 | 适用场景 |
|---|---|---|
| 演绎(必然) | 强 | 前提为真则结论必然为真 |
| 归纳(概率) | 中 | 样本→总体的统计推断 |
| 类比(启发) | 弱 | 已知相似情境迁移判断 |
| 直觉 / 断言 | 极弱 | 仅靠权威或感觉 |

**红旗:同一段混用多种推理且未声明切换 = 逻辑混乱。**

## 逻辑谬误排查

逐段扫 [logical-fallacies.md](logical-fallacies.md)。每发现一个,在段落批注中记录:`[谬误:xxx]`,并指出原文。

### 高频必查(任何定调都要查)

- 循环论证(用结论证前提)
- 诉诸权威(未给权威的资质或反例)
- 诉诸情感(用愤怒/恐惧代替证据)
- 稻草人(曲解对方观点再反驳)
- 滑坡(无依据的链式推断)
- 虚假两难(忽略中间选项)
- 因果混淆(相关≠因果)
- 幸存者偏差(只看成功案例)

## 段落连贯性额外检查

### AI / 模板化写作的段落特征(红旗)

- 每段长度异常均匀(机器生成)
- 段落间用大量过渡词("此外"、"与此同时"、"综上所述")
- 段落内反复出现排比结构
- 没有真正的反例或反对意见
- 多段是同义反复而非递进

## 输出格式

每段顶部批注格式:

```
[段落 3] 主题句: ___
论点: 单点 / 多点
推理: 演绎 / 归纳 / 类比 / 直觉
关系(与前段): 递进 / 并列 / 对比 / 回扣 / 游离 / 矛盾
谬误: [类型1, 类型2] (如无,标"无")
回扣立论: ☐ 是 ☐ 否
```

把汇总填进报告模板的 **L2 段落层(权重 30%)** 区块。
