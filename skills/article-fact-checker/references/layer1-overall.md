# L1 整体层审查清单

整体层只解决一个问题:**作者在主张什么?立论是否经得起审视?**

## 8 项必查

### 1. 立论清晰度

能否用一句话复述作者的主张?

- ☐ 是 — 复述:`________________`
- ☐ 否 — 模糊不清 / 多个主张混杂

如果立论都不能复述,后面所有论证都没有靶子,直接标红旗。

### 2. 范围边界

时间、地域、对象、条件是否明确?

| 维度 | 是否界定? | 实际范围 |
|---|---|---|
| 时间 | ☐ | |
| 地域 | ☐ | |
| 对象 | ☐ | |
| 条件 | ☐ | |

未界定的范围 = 隐含的"普遍性断言",通常是漏洞。

### 3. 立场 / 利益声明

- 作者是谁?所属机构?
- 资助方是谁?
- 有无利益相关方(商业、政治、意识形态)?

**未声明 = 不等于无利益,等于低可信度。**

### 4. 可证伪性

结论能否被反例推翻?

- ☐ 强可证伪 — 明确给出"如果 X 不成立则结论失败"
- ☐ 弱可证伪 — 模糊可证,但反例条件不清
- ☐ 不可证伪 — 无论发生什么都对(套话/信仰)

**不可证伪 ≈ 没有知识含量。**

### 5. 共识程度

结论在主流学界/业界/相关社群的位置:

- ☐ 高度共识(>90% 主流认可)
- ☐ 多数派(60-90%)
- ☐ 少数派(20-60%)
- ☐ 孤证(<20%)

立场越极端越要严查。

### 6. 替代解释覆盖

作者是否主动列出并反驳其他解释?

- ☐ 是 — 列出 ≥1 个替代解释并反驳
- ☐ 部分 — 提到但未反驳
- ☐ 否 — 只字不提反对意见(红旗)

### 7. 预测能力

结论能否做出可被未来检验的具体预测?

- ☐ 强 — 给出具体预测(时间/数量/事件)
- ☐ 弱 — 模糊趋势
- ☐ 无 — 纯解释性,无预测

### 8. 时效性

- 数据/事件是否在结论的"保鲜期"内?
- 有无引用过期数据但用来支持当前结论?

## 整体定调判断

每篇文章都要标一个**定调类型**,这影响评分权重:

| 定调 | 证据要求 | 建议权重(L1/L2/L3) |
|---|---|---|
| 学术论文 / 同行评议 | 严格 — 必须有 DOI / 数据可复现 | 25% / 25% / 50% |
| 严肃媒体 / 调查报道 | 高 — 一手来源 + 多源交叉 | 30% / 30% / 40% |
| 评论 / 专栏 | 中 — 论点清晰 + 合理证据 | 40% / 35% / 25% |
| 自媒体 / 营销文 | 严苛 — 任何未声明前提都要扣分 | 30% / 30% / 40% |
| 宣传 / 政治话语 | 极严苛 — 替代解释必查,模糊归因必查 | 35% / 35% / 30% |

## 常见红旗(任一命中即降一档整体可信度)

- 立论前后不一致(摘要和结论矛盾)
- 标题与正文主张不同(标题党)
- 隐含前提未声明却贯穿全文
- 全文找不到一个具体反例条件
- 替代解释完全缺席
- 作者身份 / 资助方不透明
