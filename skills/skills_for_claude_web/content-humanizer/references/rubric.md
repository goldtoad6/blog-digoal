# 判断层评分锚点

通读原文后，针对下面四个维度打分时，可以参考这些锚点定位大致区间，再结合实际内容
微调，不用机械对号。

## 维度一：具体细节与个性化（0-100）

- **0-39**：全篇都是"很多企业""不少用户""相关研究表明"这类不指名道姓、不给具体
  数字的泛化表述，找不到一个具体到能验证的细节。
- **40-59**：偶尔有一两个具体数字或例子，但大部分论述还是停留在概括层面，例子像是
  为了"显得具体"硬塞进去的，跟上下文论证关系不紧密。
- **60-79**：有明确的具体信息（人名/地名/时间/数字/真实场景），但整体行文仍偏向
  通用模板，个性化的"作者声音"不强。
- **80-100**：细节具体且与论点紧密咬合，能感觉到背后有一个真实经历过这件事、
  有自己观察角度的作者，而不是在转述一个通用结论。

## 维度二：情感与态度自然度（0-100）

- **0-39**：永远"一方面…另一方面…"、永远不下判断、永远四平八稳，读完不知道作者
  到底是支持还是反对、喜欢还是不喜欢。
- **40-59**：偶尔流露一点态度，但很快又回到中立陈述，态度表达像是补丁式地加上去的。
- **60-79**：有明确立场，但情绪表达方式比较单一（比如永远是"理性分析"的语气，
  没有语气的起伏变化）。
- **80-100**：能读出真实的情绪起伏——可能是吐槽、可能是兴奋、可能是犹豫不决，
  语气会随着内容自然变化，不是一种腔调撑到底。

## 维度三：结构机械度（0-100，注意这里 100 分=结构很自然，不是越机械越高分）

- **0-39**：明显的"三段论"或"总分总"模板，每个论点下面配 2-3 句车轮式解释，
  段落长度高度一致，标题/小标题对称到像是从模板里套出来的。
- **40-59**：结构总体还是工整的，但偶尔有一两处长短不一致，看得出有调整的痕迹。
- **60-79**：结构服务于内容，但仍能感觉到"先讲A，再讲B，再讲C"的清单感。
- **80-100**：内容的逻辑顺序看起来是"事情本来就是这么发展的"，不是套进了一个
  预设的框架，段落长短、论述节奏自然起伏，该详细的地方详细，该一句话带过的就
  一句话带过。

## 维度四：口语化与不完美痕迹（0-100）

- **0-39**：每句话都语法完整、用词规范，读起来像教材例句或新闻通稿，没有任何
  口语词、插话、自我修正的痕迹。
- **40-59**：偶尔出现一个口语词，但整体语域依然偏正式，口语化的部分跟周围文字
  风格不太融合。
- **60-79**：整体语域已经比较自然，有口语连接词、有适度的不完美（比如句子没那么
  工整），但还是能感觉到"被打磨过"。
- **80-100**：读起来就像一个人在认真但不刻意地说话/写字——会有插话、会有"说白了"
  "其实吧"之类的口语连接，句子结构有自然的不规整。

**重要提醒**：不是所有正式文体都应该往"口语化"方向打高分。比如法律合同、学术
论文、政府公文，本来就应该是规范书面语，这种情况下"口语化与不完美痕迹"这一项
不必强求拉满，可以在报告里说明"该文体本身对口语化没有诉求，此项不作为改写重点"，
转而把改写精力放在"具体细节"和"结构机械度"上（这两项跟文体正式程度关系不大，
正式文档同样可以写得不机械、不空洞）。

# 常见改写手法对照表

| 问题 | AI 腔示例 | 改写方向 |
|---|---|---|
| 总结腔起手 | "综上所述，这是一个值得关注的趋势" | 直接给结论："这事接下来会越来越明显" |
| 永远中立 | "一方面带来了机遇，另一方面也存在挑战" | 给出明确判断："机会确实有，但坑也不小，尤其是……" |
| 三段论模板 | "首先……其次……最后……" | 按内容本身的逻辑顺序重新组织，不强行三段 |
| 空泛主体 | "很多企业开始关注……" | 换成具体可指代的主体，或干脆去掉这层包装直接讲事 |
| 过渡词堆砌 | "此外，与此同时，不仅如此" | 删掉多余的，留一个最贴切的，或者用因果关系直接连接两句话 |
| 完美无瑕的句子 | 每句话语法工整、长度相近 | 让句子长短交替，偶尔用一个短句或反问打破节奏 |
| 商业 buzzword 堆叠 | "赋能业务增长，实现降本增效的闭环" | 换成具体说的是什么动作、带来什么实际结果 |
